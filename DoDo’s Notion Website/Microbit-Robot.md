**How To Connect a Microbit Controller To a Computer and Code It:**
1. Open https://makecode.microbit.org and log in.
2. Start new project.
3. Connect ur microbit controller to your computer by its cable.
4. Click the three dots right next to the download button on the left bottom of the screen and follow instructions.
5. You can write your code by blocks or using a coding language.
6. After writing your code press download and ur controller should be working.

**How To Connect a Microbit Controller To a Micro Servo Motor (SG90):**
1. Get 3 male to male jumper cables for each motor.
2. Get multiple crocodile cables.
3. Plug one side of the cables to the motors cable and the other end of the cables to the crocodile cable.
4. Clip the crocodile cables to the controller based on the instructions on the image below.
![Microbit introductions](Microbit%20Robot%20Project/Microbit%20intruductions.png)

**Notes and Etceteras:**
1. Make sure the cables stay intact.
2. Make sure to enable servos from extensions before coding to use servos.
3. Make sure the scaling is correct while printing.
4. Make sure to decrease filling while printing to save up on plastic usage.
5. If you're using glue make sure to get as little as you can on the servos.
6. Make sure the cables are color coded so things wouldn't get confusing while connecting them.
7. Make sure the crocodile cables are clipped to the right areas on the microbit controller.
8. Make sure to check the simulation before sending the code to your microbit controller.
